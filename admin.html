Nova Oração Oficial

Dica: comece a linha com V. para o primeiro narrador e R. para a segunda voz. Para citar outra oração salva (pessoal ou oficial), use o botão acima — ele te pergunta quantas vezes repetir (ex: [Ave Maria]{10} para uma dezena). Para uma pequena pausa entre uma oração e outra, use + Inserir pausa (ex: [pausa]{2} para 2 segundos). Em ladainhas, onde a resposta se repete sempre igual, escreva {R. rogai por nós.} numa linha sozinha para ligar a resposta automática — ela passa a aparecer depois de toda linha V. seguinte, sem precisar repetir. Use {} sozinho para desligar.
