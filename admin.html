Dica: comece a linha com V. para o primeiro narrador e R. para a segunda voz. Para citar outra oração salva (pessoal ou oficial), use + Inserir → Oração — ele te pergunta quantas vezes repetir (ex: [Ave Maria]{10} para uma dezena). Para uma pequena pausa entre uma oração e outra, use + Inserir → Pausa (ex: [pausa]{2} para 2 segundos). Use + Inserir → Leitura opcional para citar uma oração salva como texto complementar que fica oculto e fora da fala até o usuário decidir mostrar na hora de rezar. Use + Inserir → Link para inserir uma URL que aparece para abrir, mas não entra na narração. Em ladainhas, onde a resposta se repete sempre igual, escreva {R. rogai por nós.} numa linha sozinha para ligar a resposta automática — ela passa a aparecer depois de toda linha V. seguinte, sem precisar repetir. Use {} sozinho para desligar.
